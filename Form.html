Explication du Code

Ce formulaire de connexion vous redirige vers ma page GitHub lorsque l'on clique sur le bouton "Se connecter". Le script JavaScript "redirectToGitHub()" est exécuté lors de la soumission du formulaire, ce qui empêche l'envoi réel du formulaire et redirige vers l'URL indiqué.
